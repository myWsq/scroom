# Debug

The debug tool of Scroom can help you locate the problem quickly.

```js
import { createScroom, debug } from "scroom";

const sc = createScroom(...);

debug(sc)
```

## Example

<Playground>
  {() => {
    const ref = React.useRef();
    React.useEffect(() => {
      const sc = createScroom({
        target: ref.current,
      });
      const debugController = debug(sc);
      return () => {
        sc.destroy();
        debugController.destroy();
      };
    }, []);
    return <div ref={ref} style={{ height: "300px" }} />;
  }}
</Playground>

<br />

You can also provide a debug ID.

```js
debug(sc, "your-id");
```

## Important

It is not recommended to use debug tool in a production environment. It will affect performance and bundle size.
