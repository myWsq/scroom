# Usage with React

@scroom/react provides hooks to simplify usage.

<br />

## Installation

```shell
$ npm install @scroom/react
```

## Usage

```js
import { useScroom } from "@scroom/react";
import { useRef } from "react";
function App() {
  const ref = useRef();
  const sc = useScroom(ref);
  return <div ref={ref}>...</div>;
}
```
