# Usage with React

@scroom/react provides hooks to simplify usage.

<br />

## Installation

```shell
$ npm install @scroom/react
```

## Usage

```js
import { useScroom, useDebug } from "@scroom/react";
import { useRef, useEffect } from "react";
function App() {
  const ref = useRef();
  const sc = useScroom(ref);
  useDebug(sc);
  useEffect(() => {
    // can be empty
    if(sc) {
      ...
    }
  }, [sc])

  return <div ref={ref}>...</div>;
}
```
